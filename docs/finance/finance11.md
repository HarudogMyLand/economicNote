# 货币政策

现代货币政策框架力图通过科学的货币政策决策、创新的货币政策工具和畅通的货币政策传导机制达到最终目标。

**货币政策目标**：指通过货币政策的制定和实施所期望达到的最终目的，是中央银行的最高行为准则。
- 币值稳定：主要指物价稳定，也包括汇率稳定。
- 经济增长：产出水平的持续增长
- 充分就业：失业率处于自然失业率水平。
- 国际收支平衡：经常项目、资本和金融项目引起的对外货币收支基本平衡。
- 金融稳定：压力环境下，金融体系仍能正常发挥主要功能。
