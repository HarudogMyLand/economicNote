# 金融机构体系

## 章节概览

- 金融业与金融机构
- 银行业
- 证券业
- 保险业

金融机构是金融业的主体，为实体经济部门提供i内容服务并创造产值。强大的金融机构是六个强大中的核心要素之一。本章将阐述中国特色金融业、金融机构功能与作用，介绍银行业、证券业和保险业三类金融机构与行业特征。

## 金融业与金融机构

### 金融业

**金融业**是一国服务业的重要组成部分，是**经营金融商品和服务的特殊行业**，为社会经济发展提供支付、融资、投资、保障、信息等金融产品与服务。目前，中国已经形成**银行业**为主体，证券业、保险业等各行业协调发展的金融机构体系。

基本特征：

- **经营业务与对象特殊：** 经营的是“货币”和“风险”，提供金融服务、资源配置、风险管理功能
- **与实体经济关联性强**：实体投融资需要而产生，实体经济发展决定金融发展；实体投融资需要而产生，实体经济发展决定金融发展
- **高杠杆与强监管**：局部风险一旦失控就可能引发扩散

金融业三大产业：银行业、证券业、保险业

### 金融机构

**金融机构**（Financial Institution）：从事金融活动与服务的组织机构，是金融业的运作载体之一。

- 货币调控视角分：存款类金融机构、非存款类金融机构
- 业务性质视角分：商业性金融机构、政策性金融机构；
- 行业功能视角分：银行业、证券业、保险业、信托业、租赁业

### 经营模式演变

- **分业经营（Segregated）：** 银行、证券、保险“各行其道”，优点是风险隔离，缺点是效率较低。
- **混业经营（Universal）：** 全能银行模式，优点是资源共享、一站式服务，缺点是风险容易交叉传染。

### 金融机构基本功能与作用

**金融机构的主要作用：**

1. 提高全社会的资金融通效率，促进交易成本降低。
2. 提高金融资源配置效率，促进经济结构与产业结构调整。
3. 支持实体经济发展，促进国家发展战略实施。
4. 管控、降低整体金融风险，维护金融体系安全。

### 中国金融机构体系

**改革开放以来，中国已由计划经济时期的“大一统”金融体系，逐步发展成为以银行业为主体、各行业协调发展的大规模、多元化、市场化、竞争性的金融机构体系。**

1. 产业结构：以银行金融机构为主体
2. 规模结构：以大型金融机构为主体
3. 产权结构：以国有控股为主体

## 分行业运作机制

### 银行业金融机构（小重点）

特点：
- 以商业银行为主体的银行业市场结构
- 政策性金融机构与商业性金融机构相分离
- 以存贷款为主的资产负债业务结构
- 以支付结算业务为主的中间业务结构
- 产权组织形式结构以股份制为主
- 以资本管理为核心的风险管控框架

中国银行业金融机构主要有

- **存款类：** 商业银行、储蓄机构、信用社。
  - **特征：** 能创造存款货币（M2）。
- **非存款类：** 信托、租赁、财务公司。
  - **特征：** 不能吸收公众存款，是**影子银行**的主要组成部分。

存款类以存款为主要负债，以贷款为主要资产，以办理转账结算为主要中间业务，直接参与存款货币创造过程的金融机构。

非存款类金融机构则第一，不以吸收存款为主要资金来源，不直接参与存款货币的创造，只提供各种专业化的金融服务；第二，资金来源与运用方式各异，导致不同类型的非存款类金融机构的资产负债项目差异很大；第三，专业化程度高，具体业务存在较大区别；第四，业务承担的风险不同，相互之间的风险传染性相对较弱

#### 商业银行业务（重点）

商业银行业务可以分为表内业务和表外业务，其中前者计入资产负债表，占用资本；后者不计入总额，不直接占用资本。

| **业务大类** | **细分类型**           | **核心科目/业务内容**               | **逻辑与特征**                                               | **对银行的影响**                                     |
| ------------ | ---------------------- | ----------------------------------- | ------------------------------------------------------------ | ---------------------------------------------------- |
| **表内业务** | **负债业务** (来源)    | 存款、 同业借款/拆借 、银行资本     | 遵循《巴塞尔协议III》要求，资本需抵御信用、市场和操作风险。  | 决定了银行的**资金成本**和**经营规模**的上限。       |
|              | **资产业务** (运用)    | 现金资产、 贷款业务、证券投资       | **获取利润的手段**：平衡“三性”——现金保流动性，贷款保收益，证券做二级缓冲。 | 决定了银行的**盈利水平**和**风险暴露**程度。         |
| **表外业务** | **中间业务与表外创新** | 支付结算、 担保与承诺、理财与衍生品 | **轻资本运营**，基于技术、信用和劳务提供服务，不增加资产负债总额。 | 提升**非利息收入**，在不扩张规模的前提下提高 $ROE$。 |

负债业务要点解读：

- 存款：商业银行最主要、最原始的负债，例如活期、定期、储蓄
- 借款：商业银行吸收的非存款资金
- 其他负债：临时占用的客户资金
- 银行自有资本

资产业务要点解读：

- 现金资产：第一准备金，流动性强且收益低
- 贷款业务：最主要的营利性业务且需要防范风险
- 证券投资：增加收益来源，分散风险

#### 商业银行业务特点

- 信用性：基于严格的信用机制进行借贷关系
- 风险性：高杠杆和利率敏感性
- 服务性：金融业的服务性

因此，商业银行必须具有：
- 公信力
- 流动性
- 信息处理能力

的内在要求，并具有**安全性，流动性，效益性**三原则


#### 政策性银行

- 国开行、进出口银行、农发行

#### 非存款类机构（小重点）

例如信托公司，金融租凭公司，金融资产管理公司，金融消费公司等等……其共同点是：一、不吸收存款作为主要资金来源，不参与货币创造；二、资金来源各异，导致资产负债项目多样；三、专业化程度很高，业务千差万别；四、业务风险多样，感染风险低。

- 信托公司：收取手续费，代管资产，连接投资者（钱）与融资方（项目）
- 金融资产管理公司：专门收购并处理金融机构的不良资产与商业资产
- 金融租聘公司：银行系租赁公司为主，通过购买设备租给企业使用，收取租金
- 消费金融公司：为个人提供除汽车和房产以外的小额、分散消费贷款

### 证券类金融机构（重点）

**证券业金融机构**是专门从事赠券投资活动服务的金融行业，是通过各类证券、票据等债权或股权凭证，为资金供求双方直接达成资金配置提供中介服务的金融机构。

行业特征：

- 以有价证券为业务活动的载体，规范性要求高
- 业务与服务类型广泛，机构类型多，涉及面广
- 在促进有价证券投融资活动中发挥核心作用

业务运作特点：

- 风险较大
- 专业性强、要求高；
- 公开、公平和公正

证券类公司类型多样，各具特色：

- 证券公司：依法在证券市场上经营与证券相关业务的金融机构
- 证券基金投资管理公司：以金融资产为专门经营对象，以资产的保值增值为根本目的，把具有相同投资目标的众多投资者的资金集中起来，通过投资组合将资金分散投资于各种有价证券等金融工具的金融机构
- 投资咨询公司：证券资讯相关业务
- 登记结算公司
- 证券资信评级机构

**证券业金融机构通过证券发行与交易业务，为资金供求双方提供中介服务，业务专业性强，风险大，公开公平公正是其基本原则**

### 保险业

保险业金融机构：为企业和个人提供相关保险产品和服务的金融机构。

- **四大特征：** 互助性、经济性、契约性、科学性。

保险的两大类型：

- 财产保险业务：财产损失保险、责任保险、信用保险、保证保险等。
- 人身保险业务：人寿保险、健康保险、意外伤害保险等。

| **行业**   | **核心功能**                     | **主要负债（资金来源）** | **主要资产（资金去向）** | **核心原则/特征**      |
| ---------- | -------------------------------- | ------------------------ | ------------------------ | ---------------------- |
| **银行业** | 信用中介、支付结算、**货币创造** | 存款、借款               | 贷款、证券投资           | 安全性、流动性、效益性 |
| **证券业** | 直接融资中介、资产定价           | 自有资本、客户保证金     | 有价证券、做市资产       | 公开、公平、公正       |
| **保险业** | 风险分担、补偿损失               | 保费收入（准备金）       | 固定收益投资、权益投资   | 互助性、契约性、科学性 |

[^1]: 包括库存现金，央行存款，存放同业存款，在途资金
