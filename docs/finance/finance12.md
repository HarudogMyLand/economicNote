# 金融开放与全球治理

## 金融开放

金融开放的三个组成部分：

- 金融业对外开放
- 金融市场对外开放
- 资本账户开放

**金融开放需要依据本国的宏观经济、财政、金融体系、国际收支等发展状况，确定开放的程度、顺序和内容。**

### 中国金融业对外开放特点

- 1980年开始银行业的对外开放
- 千禧年开始，扩大银行业开放
- 十八大后加快银行业开放

### 资本账户开放

经常账户与资本账户下的货币可兑换

**对跨境资本进行适当管理，有助于避免本币过度升贬值和国际收支失衡等问题。**

基本原则：协同推进资本项目可兑换、金融服务业开放、利率市场化改革和汇率形成机制改革，长期采取渐进式、审慎的方式开放资本账户.

开放顺序：

- 内容：先开放直接投资，后开放证券投资和金融衍生品；
- 方向：先开放资本流入，后开放资本流出；
- 期限：先开放长期资本交易，后开放短期资本交易；
- 主体：先开放法人机构，后开放个人。


开放状态：按照IMF资本项目交易分类标准下的40个子项来看，我国可兑换和部分可兑换的项目超过90%。

### 国际收支平衡表

记录境内外之间全部经济交易的统计表

国际收支平衡表三大账户：
- 经常账户：货物与服务账户、初次收入账户、二次收入账户
- 资本与金融账户
  - 资本账户：非生产非金融资产的取得和处置
  - 金融账户：金融资产和负债的收买和放弃，包括直接投资、证券投资、金融衍生品、其他投资、储备资产
