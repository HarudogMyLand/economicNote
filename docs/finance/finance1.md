# 绪论与金融体系

## 金融的内涵

- 金融是货币与信用的融合体，是与货币信用相关的所有活动的集合
- 金融学是研究各类经济主体金融活动和金融体系各部分相互关系与整体运行规律的科学。

金融的构成要素主要有：

1. **货币**流通及其管理
2. **汇率机制与外汇**
3. **信用活动**引起的债权债务或权益关系
4. 信用活动引起的**利率**作用
5. **金融机构**为载体的金融产业
6. 开放经济下的**国际金融关系**
7. 金融风险与监督
8. 金融安全与审慎
9. **央行**制度与业务
10. **货币供求与均衡**
11. **货币政策与宏观调控**
12. 金融发展理念及路径选择

金融学课程也是按照这个顺序组织的教学思路。

### 金融体系

金融体系是指各种要素在一定制度法规框架下组成的系统性整体，主要包括

- 金融工具机器价格体系
- 金融机构体系
- 金融市场体系
- 金融监管体系
- 政策调控体系
- 金融基础设施服务体系

### 本质与功能（重点）

**金融本质是服务**，坚持以人民为中心的价值取向，社会主义市场经济体制下的金融和信誉关键是为**实体经济服务**。

金融的本质是通过**金融功能**来实现的，金融功能通过金融活动发挥出来。

1. 提供货币充当**支付中介**
2. 满足投融资充当**信用中介**
3. 资源配置充当**相关管理者**
   1. 信息管理者
   2. 风险管理者
   3. 资产管理者

### 地位与作用

1. 金融是国民经济的血脉
2. 金融在现代经济中发挥核心作用
3. 金融是“国之大者”
   1. 核心竞争力
   2. 现代化建设
   3. 金融安全是国家安全重要组成

### 金融强国要求

- **六个强大：“强大的货币，强大的央行，强大的金融机构，强大的国际金融中心，强大的金融监管，强大的金融人才队伍。”**
- 五大文章：科技金融、绿色金融、普惠金融、养老金融和数字金融
