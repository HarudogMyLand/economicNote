信用是抽象的金融要素，蕴含在金融活动的交换中，是最普遍的经济金融活动，是经济主体之间调节余缺的手段。

## 信用是什么

社会学上我们将信用称作诚信。而经济学意义上，**信用是一种借贷关系**，以信任为基础，偿还和支付利息为条件的借贷行为。[^1]借贷，就是资金流入行为。借贷行为主要分为两种，**实物**借贷与**货币**借贷。

**信用建立在诚信之上，偿还与收回本金，支付与收取利息是借贷关系确立的必要条件。保持所有权和获得收益权是权益投资的必要条件。**

**社会主义市场经济是信用经济，法治意识、契约精神、守约观念是现代经济活动的重要意识规范，也是信用经济、法治经济的重要要求。**

### 信用的产生

**信用的起源是借贷意愿，私有制是信用的基础，产权制是信用的催化剂。**

只有在财产与所有者息息相关的私有财产制度下，借贷作为经济行为才有现实性。产权的发展，清晰界定，流转保护制约了信用履行能力。

### 信用的形成

- 早期货币与信用

货币为信用规模拓展了形态，信用为货币形态拓展了领域。然而在信用货币制度与现代市场经济出现之前，两者范畴一直相对独立。

- 现代货币与信用

随着货币与信用彻底挂钩，在债权债务上，进入货币与信用等同的时代。现代金融随着信用与货币的结合诞生。**金融，即是以货币为载体的借贷活动。**

- 高利贷

极高利息为基本特征、高浮动利率为重要特征的借贷活动。极高利率、高风险、高成本。

生产力低下导致借贷需求迫切；放贷者（地主/钱庄）垄断资金；借款人违约风险极高。

资本原始积累的手段，但阻碍现代工业发展；最终被现代银行信用取代。[^2]

**简而言之，即借贷资金供求矛盾，贷者垄断，高风险高成本综合导致了高利贷利率。**

在资本主义初期阶段，高利贷促进了资本原始积累，同时也某种程度上阻碍了资本家的实业工作。随着金属货币体系全面崩溃，旧高利贷者的垄断地位与乏借贷资本的现实被彻底打破，高利贷体系也随之崩溃。

## 信用活动

### 市场中的投融资活动（小重点）

- 实物投资与金融投资

  前者是购买经营性资产或各类收藏型物品以期获利；后者是金融投资，指投资金融产品，以期再风险承受范围内获益。

- 间接融资与直接融资

  前者是盈余者通过**金融机构**间接向赤字者融资；后者是盈余者通过赤字方的**所有权凭证或债权债务**直接融入资金。

- 债权融资与股权融资

  前者是由债权债务的融资，只有收益权；后者基于股权的融资，股东可以参与生产决策。信用的所有权和经营权实质分离，这是股份制产生和发展的基础。

### 信用形式与金融产品（小重点)

**社会融资规模是金融部门服务非金融部门的信用规模，是指一定时期末实体经济从金融体系获得的资金余额**

### 信用形式

国民经济中居民个人、非金融企业、政府、金融机构与国外五个部门的资金余缺并存是市场经济中信用关系存在的基础。

#### 企业及其商业信用

企业是最大的债权人，也是最大的债务人。

**债权融资**是企业接受信用最主要的形式，形成企业的债务；**股权融资**是企业接受信用的重要形式，形成企业的资本。具体有：

- **商业信用**：最基本的债权融资，无需中介。商业票据等凭证可以用来表明双方债权债务关系。
- 银行贷款：最常见的债权融资，银行作为中介。
- 发行债券：企业发行合法有价证券，向债权人承诺还本付息的债权。
- 股权融资：股票融资、增资扩股是典型股权融资。股票分为优先股与普通股
- 信托与融资租聘
- 其他：例如期权、衍生品等等

#### 银行信用

银行等机构作为债权债务中介参与借贷活动。银行或者其他金融机构以货币形态提供的信用。是一种间接信用。

银行金融工具：

- 贷款：依法按一定利率将贷款发放给资金需要者
- 银行承兑汇票
- 证券投资

#### 政府信用

政府作为债权人或债务人的借贷活动。

- 中央政府债权，即国债：一国中央政府为弥补财政赤字或筹措建设资金而发行，分为短中长期。
- 地方债券：地方政府发行的债券，分为一般债券和专项债券
- 担保债权或政府支持债券

#### 消费信用

居民部门所接受的信用，企业、银行和其他金融机构向消费者提供的用于生活消费的信用。

例如储蓄存款(居民向银行提供信用)或金融投资等。

#### 国际信用

一切跨国的借贷关系和借贷活动。

- 出口信贷：出口国政府为了支持扩大本国产品出口，通过利息补贴和信贷担保鼓励本国银行向出口商或购买本国商品的外国进口商的中长期信贷。
- 国际商业银行贷款：一些大型商业银行向外国政府及其所属部门、私营工商企业或银行提供的中长期贷款
- 外国政府贷款：他国政府利用国库资金向本国政府的贷款
- 国际金融机构贷款

## 信用体系构建

### 市场经济信用体系

社会主义市场经济的平稳运行离不开健康的信用秩序，良好的信用秩序是支撑信用关系良性发展的前提条件。

### 信用统计（重点）

对信用进行统计可以了解其结构特征，分析经济运行状况。社融（资产端）衡量金融机构把钱“借出去”了多少（实体经济拿到了多少钱）。**M2（负债端）衡量银行系统“存进来”了多少钱（实体经济手里有多少钱）。

信用的统计和分析可以从投资角度和融资角度出发，通常对**信用活动形成的融资规模**进行统计。在我国，使用**窄口径**（人民币贷款）和**广口径**（社会融资规模）。

中国人民银行统计的社会融资规模包括存量和增量，前者指**一定时期末**从金融体系获得的资金**余额**；后者指的是**一定时期内**从金融体系获得的资金**总额**。

社会融资规模可以参考以下示例！

| **类别**         | **包含的具体项目**                                           |
| ---------------- | ------------------------------------------------------------ |
| **1. 表内融资**  | 人民币贷款、外币贷款（银行资产负债表里的正规军）             |
| **2. 表外融资**  | 委托贷款、信托贷款、未贴现银行承兑汇票（所谓的“影子银行”部分） |
| **3. 直接融资**  | 企业债券、非金融企业境内股票融资（企业直接去市场找钱）       |
| **4. 其他/政府** | **政府债券**（包括国债、地方债）、保险、资性房地产，非金融境内股票融资等 |
